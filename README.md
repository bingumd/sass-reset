# @bingumd/sass-reset

Small sass reset with options.

## :hammer: Installation

##### Npm:
```sh
npm install @bingumd/sass-reset
```

##### Yarn:
```sh
yarn add @bingumd/sass-reset
```

## :gear: Usage

##### Default:
```scss
@use '@bingumd/sass-reset';
```

##### With options:
```scss
@use '@bingumd/sass-reset' with (
    $body-line-height: 1.8,
);
```
